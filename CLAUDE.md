# compulsory session logging

at session end, write a detailed note to Obsidian vault via MCP. always. no exceptions.

- path: `Nora/sessions/YYYY-MM-DD-description.md`
- frontmatter: date, status, tags (nora/session, nora/handoff)
- content: work done, decisions, pending items, mistakes, repo state, branches, PRs
- also update `Nora/session-context.md` with active state

do not wait for reminders. if context is about to compact, write the handoff first.
