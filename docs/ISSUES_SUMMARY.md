**Grouped Issues**

Below are the common issue groups I found along with representative issue numbers and links.

- **1) Playback / Decoder Errors (FFmpeg, demuxer, CODE 3/4)**
  - Representative: #447, #427, #405, #404, #448, #398, #452
  - Symptoms: CODE 3/4 errors, DEMUXER_ERROR_COULD_NOT_OPEN, PIPELINE_ERROR_DECODE, format errors for FLAC/MP3/ALAC.
  - Recommended action: collect sample files, system logs, and FFmpeg demuxer traces; prioritize reproducible cases.

- **2) Import & Library / Bulk-import Crashes**
  - Representative: #406, #458, #433, #417
  - Symptoms: app crash during bulk import, libvips/sharp assertion, AppImage/package import failures, directories not indexed.
  - Recommended action: reproduce with a small subset, capture main process logs, and isolate libvips/sharp usage during artwork generation.

- **3) Linux Packaging / Runtime Issues**
  - Representative: #458, #387, #433, #417
  - Symptoms: AppImage / packaged release specific failures, chrome-sandbox permission issues, xdg-dialog selection quirks.
  - Recommended action: platform-specific bug triage, add packaging notes, and test AppImage / deb / rpm artifacts.

- **4) macOS Specific Issues**
  - Representative: #456, #454, #399, #400
  - Symptoms: titlebar/menu overlap, oversized app icon, gatekeeper refusing ARM builds, installer/run problems.
  - Recommended action: gather repro steps for each macOS version and verify notarization / packaging process.

- **5) UI / UX & Feature Requests**
  - Representative: #461, #451, #449, #434, #430, #432, #423, #402
  - Items: lyrics panel, card view, collapsible settings, sleep timer, now-playing/fullscreen behavior, equalizer improvements.
  - Recommended action: triage into roadmap / milestone buckets (some already in `Nora v4.0.0`).

- **6) Playlist Ordering & Reordering**
  - Representative: #455, #420, #434
  - Symptoms: missing drag-to-reorder, persistent per-playlist sort expectations.
  - Recommended action: decide expected behavior (persistent per-playlist order vs global sort) and implement ordering APIs.

- **7) Translations & Documentation**
  - Representative: #401, #416
  - Items: pt-BR and zh-TW translation files submitted; documentation requests for macOS build behavior.
  - Recommended action: merge verified translations and update contributor docs.

- **8) Renderer / JavaScript Errors**
  - Representative: #413, #395
  - Symptoms: recurring JS exceptions that pause playback or show repeated error modals.
  - Recommended action: attach dev logs, reproduce in dev build, and add defensive error handling.

- **9) Misc / Integrations**
  - Representative: #393 (Last.fm scrobble), #389 (m3u8 import), #412 (metadata edits causing corruption)
  - Recommended action: small focused fixes or guidance for contributors.
